# Gammal Tech Market Research - Health Apps

## Overview
This research contains 50 real negative reviews for the **Sehhaty (صحتي)** app collected between 2025 and 2026.

## Key Findings (Pain Points):
1. **Technical Stability:** Frequent crashes after 2026 updates.
2. **Telemedicine Issues:** Long wait times for "Immediate Consultation" (sometimes days).
3. **Data Sync:** Failure to sync steps in "Walk 30" challenges.
4. **UX Friction:** Users forced to reinstall the app to see new medical reports.

## File Structure
- `sehhaty_negative_reviews.xlsx`: The raw data with categories.
